Vue

每个Vue应用都需要通过实例化Vue来实现。

var vm=new Vue({
    el:"#id",
    data:{
        username:"panhao"
    },
    method:{
        details:function(){
            return this.username;
        }
    }
})
  • el :el元素用来指定DOM元素的ID；
  • data :用于定义属性；
  • methods ：用于定义的函数；
  • 双大括号:用于输出对象的属性和函数的返回值；

当一个Vue实例被创建的时候，它向Vue响应式系统中加入其data对象中所有的属性。当这些属性发生变化的时候，html视图也会发生变化。

  1. ==当时数据在外部引用的时候，Vue的数据和外部的数据始终是恒等的==。
var data={
    username:"panhao",
    password:"123456"
}

var vm=new Vue({
    el:"#id",
    data:data,
    methods:{}
})

当data中的数据发生变化的时候，Vue中的数据也随之变化，同理当Vue中数据发生变化的时候，data中的数据也会发生变化。

1、模板语法

模板

影响DOM元素或者模板的选项有el和template，属性replace和template需要一起使用。

  • el ：为字符串类型，DOM元素或者函数。其作用就是为实例提供挂载元素。一般使用CSS选择符。
  • template：为字符串类型。默认会将template值替换挂载元素，并合并挂载元素和模板根节点的属性（如果属性具有唯一性，类似id，则以模板根节点为主）。
<div id="app">
    <p>
        123123
    </p>
</div>
<script id="tpl" type="x-template">
    <div class="tpl">
        <p>This is a template</p>
    </div>
</script>
var vm=new  Vue({
    el:"#app",
    template:"#tpl"
})

数据以及方法

  • 使用data元素设置数据；
  • 使用methods元素设置方法；
生命周期

Vue 的实例在创建的时候有一系列的初始化步骤，例如建立数据观察、编译模板，创建数据绑定等。

var vm=new Vue({
    el:"#app",
    creater:function(){
        console.log("创建的时候调用")
    }
})

Vue的生命周期示意图：

Vue的强大的生命周期的方式，让动画的过渡有了更好的方式。

2、数据绑定

数据绑定语法
var vm=new Vue({
    el:"#app",
    data:{
        id:1,
        index:0,
        name:'Vue',
        avatar:"https://www...",
        count:[1,2,3,4,5],
        names:["Vue1.0","Vue2.0"],
        items:[{
            {name:'Vue.0',version:"1.0"},
            {name:'Vue.1',version:"2.0"}  
        }]
    }
})

文本插值

数据绑定的最基础的形式就是文本插值，使用的双大括号表示文本插值：

<span>Hello {{name}}</span>

模板语法支持单次插值，即首次赋值之后在更改vm的属性值不会引起DOM的变化：

<span v-once="name" >{{name}}</span>

HTML

Vue同样可以插入HTML语句，使用方式和文本插值类似，不过需要用到v-bind指令：

<span v-bind:id="'id-'+id"></span>
<!--
或者简写为<span :id="'id-'+id"></span>
此时绑定的是span元素的id属性，JQuery的获取方式任然是$("#id-"+vm.id)
-->

绑定表达式

一段绑定表达式可以由一个简单的JavaScript表达式和可选的一个或者多个过滤器构成：

<span>{{index+1}}</span>
<span>{{index=0?'a':'b'}}</span>
<span>{{name.split('').join('|')}}</span>

每个绑定中只能包含单个表达式，并不支持JavaScript语句，否则Vue.js就会抛出warnning异常，并且表达式里面不支持正则表达式，如果需要进行复杂的交换，可以使用过滤器或者计算属性进行处理

过滤器

Vue允许在边大师后面添加可选的过滤器，以管道符|指示，同时也支持多个过滤器的链接使用：

<!--<span>{{name|uppercase}}</span>
<!--Vue将name的属性值传递给Uppercase这个内置的过滤器，返回字符串的大写-->
<span>{{name|filterA|filterB}}</span>-->

还可以向过滤器中传入多个参数：

{{name.split('') | limitBy 3 1}}
<!--过滤器limitBy可以接受两个参数，第一个是设置显示个数，第二个参数为可选，指从开始元素的下标-->

值得注意的是Vue.js的内置过滤器在2.0版本已经被移除了，取代的是每个区域更专业的库来解决。

过滤器 Vue.js 1.0使用形式 Vue.js 2.0使用形式 描述
uppercase `text uppercase` text.toUpperCase
capitalize `text capitalize` text[0].toUpperCase+text.slice(1)
lowercase `text lowercase`
currency 将数字转化成货币符号，并且自动添加数字分节号
pluralize 字符串复数化
json 与JSON.stringfy()作用相同
debounce `<input v-on:kepup=”onkeyup” debounce 500>`
limitBy `<name.split(‘’) limitBy m n`
filterBy `<p v-for=”name in names filterBy ‘1.0’”>

`
orderBy `<p v-for=”item in items orderBy ‘name’ -1”>

`

指令

Vue也提供指令这一概念，可以理解为当表达式的值发生改变的时候，会有一些特殊的行为作用到绑定的DOM上。指令通常会直接书写的HTML元素中，而为了有别于其他的属性，Vue.js指令通常是带有v-前缀的。

  • 参数

    <img v-bind:src="avatar" />

  • 修饰符

    <button v-on:click.stop='doClick'></button>
计算属性

在Vue项目中，展示数据的时候总是要经过一些处理。为了避免在模板中加入过重的业务逻辑，保证模板的结构清晰和可维护性。

<span>{{fullName}}</span>
var vm=new Vue({
    el:"#app",
    data:{
        firstName:"Stephen",
        lastName:"Curry"
    },
    computed:{
        fullName:function(){
            return this.firstName+"-" +this.lastName;
        }
    }
})

Setter

在处理商品的价格的时候，前端需要使用￥1.00展示给用户，但是后端存储的存储数据的形式必须为100，此时用到Setter属性的时候会更便利： 

var vm=new Vue({
    el:"#app",
    data:{
        cents:100
    },
    computed:{
        price:{
            set:function(newValue){
                this.centens= newValue*100;
            },
            get:function(){
                return (this.cents/100).toFixed(2);
            }
        }
    }
})
  • 将cents元素转换成price元素，后端传入的cents在前端转换成price，此时改变price的值也会同时改变cents的值。例如：将price改成2，￥2.00形式，cents的值就为200。这样既不影响前端的显示，也不影响后端的存储。
表单控件

Vue.js提供v-model的指令对表单元素进行双向数据绑定，在修改表单元素值的同时，实例vm中对用的属性值也同样更新，反之亦然。

  1. 绑定输入框

    <input type="text"  v-model="username" />

  2. 绑定Radio

    <input type="radio" v-model="gender"  value="男"/>

  3. 绑定checkbox

    绑定checkbox分为两种情况：一种是，使用单个的checkbox，这个时候返回的值就是boolean类型的；另一种是选择多个checkbox，此时绑定的数据类型必须为数组。

    <input type="checkbox" v-model="checked" />我已经阅读了协议
    <input type="checkbox" v-model="multiChecked" >篮球
<input type="checkbox" v-model="multiChecked" >乒乓球
<input type="checkbox" v-model="multiChecked" >羽毛球

  4. 绑定select元素

    同CheckBox元素一样，Select元素也分很两种，一种是单选，一种是多选；

    <select v-model="selected">
        <option>A</option>
        <option>/</option>
        <option>/</option>
</select>
    <select v-model="multiSelected">
    <option>A</option>
    <option>B</option>
   <option>C</option>
</select>

    ==以上四种数据的绑定除了多选需要绑定的为数组除外，其他的初始值都可以为空，但是也可以设置初始值：==

    var vm = new Vue({
        el: "#container",
        data: {
            name: "",
            gender: "男",
            multiChecked: [],
            checked: '',
            selected: '湖北省',
            multiSelected: []
        }
    })

  5. value绑定

    表单空间的值同样可以绑定在Vue实例的动态属性上，使用v-bind指令实现。

    <input type="checked" v-model="checked" v-bind:true-value="a"  v-bind:false-value="b" />

    表示在绑定了单选框的值之后，如果checked的值为true，那么checked的中就跟a的值相同，如果checked的值为false就和b的值相同。

  6. 参数特性

    Vue.js为表单空间提供了一些参数，方面处理某些常规的操作：

    lazy

    默认情况下，v-model在input事件中同步输入框的值与数据，加入lazy属性之后从会改到在change时间中同步。(加上lazy属性就是在输入框失去焦点的时候获得到值，如果不加就是同步获取值)

    <input v-model.lazy="query" />

    number

    会将用户输入转为Number类型，如果原值转换结果为NaN则返回原值。

    <input v-model.number="age" />
Class与Style绑定

可以直接使用v-bind对这两个属性进行绑定：

<p v-bind:style="style"></p>

然后通过修改vm.style的值对元素的样式进行修改。但是这样的会大大加大出错的概率。所以Vue.js提供了更好的方式来绑定这两个属性。

Class绑定

绑定Class的数据类型可以是数组，可以是对象：

  • 对象

    <div class="tab" v-bind:class="{'active':active,'unactive':!active} " />
    var vm=new Vue({
    el:"#app",
    data:{
        active:true
    }
})

    渲染的结果为<div class="tab active"></div>

  • 数组

    <div v-bind:class="[ClassA,ClassB]" />
    data:{
    classA:'class-a',
    classB:'class-b'
}

    渲染的结果为：<div class="class-a class-b"></div>

    也可以是用三元运算符渲染：

    <div v-bind:class="[isB?ClassA:ClassB]"></div>

    如果vm.isB的值为true，渲染结果为：<div class="class-a"></div>；值为false的时候，渲染的结果为：<div class="class-b"></div>

内联样式绑定

绑定Style的数据类型同样可以是数组，可以是对象：

  • 对象

    <div v-bind:style="alertStyle" />
    data:{
    alertStyle:{
        color:'red',
        fontSize:'15px'
    }
}

    此外，对象绑定的形式还支持单个属性绑定。

    <div v-bind:style="{font-size:alertStyle.fontSize,color:'red'}"

  • 数组

    <div v-bind:style="[styleA,styleB]" />

    Vue允许将多个样式的对象绑定到统一元素上。

模板渲染

条件渲染

Vue的条件渲染提供了v-if、v-show、v-else这几个指令来说明模板和数据之间的逻辑关系。

v-if、v-else

v-if和v-else的作用是根据数据值来判断是否输出该DMO元素，以及其包含的子元素：

<div v-if="yes" >yes</div>
<div v-else>no</div>

如果vm.yes的值为true，模板引擎就会编译这个DOM节点，输出<div>yes</div>

值得注意的是：v-else必须和v-if连在一起的时候才能够起作用

v-show

也是条件渲染的一种指令，同样是和v-if一样可以搭配v-else使用，但是与v-if的区别在于：v-if控制的是渲染或者不渲染，v-show控制的是显示不显示，display的属性值。

列表渲染

v-for指令主要用来做列表渲染，将根据接受到的数组重复的渲染v-for绑定的DOM元素及其内部的子元素，并且可以用过设置别名的方式，获取数组内数据渲染到节点中。

<ul>
    <li v-for="item in items">
        <h3>
            {{item.title}}
        </h3>
        <p>
            {{item.description}}
        </p>
    </li>
</ul>
var vm = new Vue({
　el : '#app',
　data: {
　　items : [
　　　{ title : 'title-1', description : 'description-1'},
　　　{ title : 'title-2', description : 'description-2'},
　　　{ title : 'title-3', description : 'description-3'},
　　　{ title : 'title-4', description : 'description-4'}
　　]
　}
});
  • 其中items为属性名，item为别名，可以用过item来获取当前数组遍历的每个元素。

v-for也内置了$index变量，可以在v-for指令调用的期间内，输出当前数组元素的索引。另外我们也可以自己制定索引的别名

<li v-for="(index.item) in items">{{index}}-{{$index}-{{item.title}}}</li>

需要注意的是Vue.js中数组的原生方法进行了封装，所以在改变数组的时能触发视图更新，但是以下两种方式是无法更新视图的：

  1. 通过索引直接修改数组元素，例如：vm.items[0]={title:"title-changed"}

    此时的解决办法通过Vue的set办法去改变：vm.items.set(0,{title:"title-changer"})或者vm.items.set("items[0]",{title:"title-changer"})

  2. 无法直接修改数组的长度，例如vm.items.length=0;

v-for同样接受单个整数和对象(接受对象的时候，使用的就是key-value值)

事件绑定与监听

与传统的监听不同的是，借助v-on指令可以在模板内绑定事件监听器。

方法以及内联语句处理器

<button v-on:click="say">say</button>
var vm = new Vue({
　el : '#app',
　data: {
　　msg : 'Hello Vue.js'
　},
　methods : {
　　say : function() {
　　　　alert(this.msg);
　　}
　}
});

Vue.js 也提供了 v-on 的缩写形式，改写为 <button @ click='say'>Say</button>，这两句语句是等价的。

除了直接绑定 methods 函数外，v-on 也支持内联 JavaScript 语句，但仅限一个语句。

<button v-on:click="sayFrom ('from param')">Say</button>
var vm = new Vue({
　el : '#app',
　data: {
　　msg : 'Hello Vue.js'
　},
　methods : {
　　sayFrom: function(from) {
　　　alert(this.msg + '' + from);
　　}
　}
});

获取原生 DOM 事件对象

<button v-on:click="showEvent">Event</button>
<button v-on:click="showEvent($event)">showEvent</button>
var vm = new Vue({
　el : '#app',
　methods : {
　　showEvent : function(event) {
　　console.log(event);
　　}
　}
});

同一元素上也可以通过v-on 绑定多个相同事件函数，执行顺序为顺序执行，例如:

<div v-on:click="sayFrom('first')" v-on:click ="sayFrom('second)">

修饰符

Vue.js 为指令v-on提供了多个修饰符，方便我们处理一些 DOM 事件的细节，并且修饰符可以串联使用。

  1. .stop: 等同于调用 event. stopPropagation()。
  2. .prevent: 等同于调用 event.preventDefault()。
  3. .capture: 使用 capture 模式添加事件监听器。
  4. .self: 只当事件是从监听元素本身触发时才触发回调。
<a v-on:click.stop='doThis'></a>
<form v-on:submit.prevent="onSubmit"></form>
<form v-on:submit.stop.prevent="onSubmit"></form>
<form v-on:submit.stop.prevent></form>

除了事件修饰符之外，v-on 还提供了按键修饰符，方便我们监听键盘事件中的按键。

<input v-on:keyup.13="submit"/>
<!--监听 input 的输入，当输入回车时触发 Submit 函数（回车的 keycode 值为 13），用于处理常见的用户输入完直接按回车键提交）-->

Vue.js 给一些常用的按键名提供了别称，这样就省去了一些记 keyCode 的事件。全部按键别名为：enter、tab、delete、esc、space、up、down、left、right。

Vue.js 2.0 中可以直接在 Vue.config.keyCodes 里添加自定义按键别名，无需修改v-on指令，例如：Vue.config.keyCodes.f1 = 12。
